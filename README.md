# RoutineReset Assets

箱庭ジオラマ画像 22枚を配信するためのアセットリポジトリ。

- 命名規則: `diorama-frames/<frame-id>.png`
- 配信 URL 例: `https://raw.githubusercontent.com/kotamzmr/RoutineReset-Assets/main/diorama-frames/day07_tanuki_arrives.png`
